# Examples

시뮬레이터 사용 예제들입니다.

## 파일 설명:

- **example1.py** - 메인 예제 (LLaMA-70B, 고부하 테스트)
- **example.py** - 기본 예제
- **communication_example.py** - 통신 전략 비교
- **detailed_breakdown_example.py** - 상세 성능 분석
- **gpu_comparison_example.py** - GPU 비교
- **performance_prediction_example.py** - 성능 예측

## 실행:
```bash
cd examples
python3 example1.py
```

각 예제는 독립적으로 실행 가능합니다.
